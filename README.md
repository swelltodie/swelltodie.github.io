#### 3. 统计 IV 号染色体上各类 feature (1.gtf 文件的第 3 列，有些注释文件中还应同时考虑第 2 列) 的数目，并按升序排列？

结果如下：

```bash
stop_codon      6692
start_codon     6700

transcript      7126
exon    7553
```

使用的代码如下：

```bash
# 先去除了前5行注释信息，从第6行开始分类统计数目
zcat 1.gtf.gz | tail -n +6 | awk '{x[$3]++} END{for(i in x) print(i "\t" x[i])}' | sort -n -k 2
```
